Lists

Ordered Lists

Basic ordered list

Uses the <ol> tag. Each item is automatically numbered, with items added using <li> tags

  1. first entry
  2. second entry
  3. third entry
  4. fourth entry

Nested ordered list

Nesting ordered lists is possible, but without css, the numbering scheme is up to the browser.
  1. first entry
  2. second entry
    1. first sub-entry
    2. second sub-entry
  3. third entry
    1. first sub-entry
      1. first sub-sub-entry
      2. second sub-sub-entry
    2. second sub-entry
    3. third sub-entry
  4. fourth entry

Unordered Lists

Basic unordered list

Uses the <ul> tag. Each item is a bullet point, with items added using <li> tags

  • first entry
  • second entry
  • third entry
  • fourth entry

Nested ordered list

Nesting unordered lists is possible, but without css, the bullet scheme is up to the browser.
  • first entry
  • second entry
    • first sub-entry
    • second sub-entry
  • third entry
    • first sub-entry
      • first sub-sub-entry
      • second sub-sub-entry
    • second sub-entry
    • third sub-entry
  • fourth entry

Mixed

When creating nested lists, can intermix <ol> and <ul> at will.

  1. first entry
  2. second entry
    • first sub-entry
    • second sub-entry
      1. first sub-sub-entry
      2. second sub-sub-entry
    • third sub-entry
      • first sub-sub-entry
      • second sub-sub-entry
  3. third entry
    1. first sub-entry
      • first sub-sub-entry
      • second sub-sub-entry
    2. second sub-entry
      1. first sub-sub-entry
      2. second sub-sub-entry
    3. third sub-entry
  4. fourth entry
