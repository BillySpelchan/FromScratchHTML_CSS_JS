HTML tables

Basic table

Your basic table uses the <table> tag to indicate there is a table. Rows of table created using <tr> tags which then contain <th> for table heading cells (such as along side/top) or <td> for regular table cells

Making columns span multiple columns can be done using the colspan attribute in the <th> or <td> attribute but it is up to you to track the columns.
